How can we use a defined typeface?
it's easier than you think! just specify your desired font variation (style, weight, display)

this is a text paragraph

 
            
              .example {
                font-family: 'CustomFont';
                font-weight: 100;
              }

this is a text paragraph

 
            
              .example {
                font-family: 'CustomFont';
                font-weight: 300;
                font-style: italic;
              }
