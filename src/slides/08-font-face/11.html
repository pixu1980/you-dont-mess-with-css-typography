How can we define a typeface?
@font-face allows us to create our custom font-family
    
      ...
      
      @font-face {
        font-family: 'CustomFont';
        src: url('path/to/CustomFont-Regular.woff2') format('woff2'),
          url('path/to/CustomFont-Regular.woff') format('woff');
        font-style: normal;
        font-weight: 400;
      }
      
      /* font style variation */
      @font-face {
        font-family: 'CustomFont';
        src: url('path/to/CustomFont-Regular-Italic.woff2') format('woff2'),
          url('path/to/CustomFont-Regular-Italic.woff') format('woff');
        font-style: italic;
        font-weight: 400;
      }
      
      ...
(NOTE: be careful, use only fonts you are licensed to use)
