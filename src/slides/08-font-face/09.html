How can we define a typeface?
@font-face allows us to create our custom font-family
    
      ...
      
      body {
        font-family: 'CustomFont', FallbackFont, sans-serif;
        font-size: 1.6rem;
        font-weight: 400;
        line-height: 2rem; 
      }
      
      @font-face {
        font-family: 'CustomFont';
        src: url('path/to/CustomFont-Regular.woff2') format('woff2'),
          url('path/to/CustomFont-Regular.woff') format('woff');
        font-style: normal;
        font-weight: 400;
      }
      
      ...
by specifying the same font-family in every @font-face definition
we're defining a typeface
