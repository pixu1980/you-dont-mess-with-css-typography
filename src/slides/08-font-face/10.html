How can we define a typeface?
@font-face allows us to create our custom font-family
    
      ...
      
      @font-face {
        font-family: 'CustomFont';
        src: url('path/to/CustomFont-Regular.woff2') format('woff2'),
          url('path/to/CustomFont-Regular.woff') format('woff');
        font-style: normal;
        font-weight: 400;
      }
      
      ...
so a single @font-face definition combines a font glyphs file, a font-style and a font-weight
