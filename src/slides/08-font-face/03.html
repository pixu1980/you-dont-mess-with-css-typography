What does font mean?
it's a single size, weight and style configuration of a typeface
    
      html {
        /* 
          16px/10px = 0.625, expressed in em
          makes the font-size reactive to browser settings
          and makes 1rem = 10px,
          allowing to use rems as a simple division base-10 value
        */
        font-size: 0.625em;      
      }
      
      body {
        font-family: Verdana, sans-serif;
        font-size: 1.6rem;
        font-weight: 400;
        font-style: normal;
        line-height: 2rem; 
      }
