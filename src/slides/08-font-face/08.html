How can we define a typeface?
@font-face allows us to create our custom font-family
    
      html {
        /* 
          16px/10px = 0.625, expressed in em
          makes the font-size reactive to browser settings
          and makes 1rem = 10px,
          allowing to use rems as a simple division base-10 value
        */
        font-size: 0.625em;      
      }
      
      body {
        font-family: 'CustomFont', FallbackFont, sans-serif;
        font-size: 1.6rem;
        font-weight: 400;
        line-height: 2rem; 
      }
      
      ...
Browsers will try to use the font you specified first (CustomFont, in this case).
If CustomFont is not available, browsers will fallback fonts in the list from left to right.
